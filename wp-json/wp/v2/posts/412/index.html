Before we get started with TASM installation on a Windows machine, let\u2019s get some knowledge about TASM and learn what it will do for us.<\/p>\n

Turbo Assembler (TASM) a small 16-bit computer program which enables us to write 16 bit i.e. x86 programming code on 32-bit machine. It can be used with any high level language compliers like GCC compiler set to build object files. So that programmers can use their daily routine machines to write 16-bit code and execute on x86 devices.<\/p><\/blockquote>\n

So let get started with installation steps. Before we get started to make sure that you have the following software. If not, you can download it by clicking on it.<\/p>\n

Requirements<\/b>.<\/p>\n

1. DosBox software.<\/a><\/p>\n

2.  TASM installation files<\/a>.
\n
\nSteps<\/b><\/p>\n

1. First, install DOSBox software on your system.<\/p>\n

<\/a><\/p>\n

2. Just run the downloaded setup file to install as just like other software you install.<\/p>\n

<\/a><\/p>\n

3. Then extract the TASM zip file.<\/p>\n

4. After that copy that extracted the folder to C drive of your system.<\/p>\n

<\/a><\/p>\n

5. Now we have to mount our C drive to DosBox so that we can use our TASM libraries there.<\/p>\n

6. To do that go to the following path on your PC<\/p>\n

For 32-bit PC<\/b><\/p>\n

C:\\Program Files\\DOSBOX-0.74<\/p>\n

For 64-bit PC<\/b><\/p>\n

C:\\Program Files (x86) \\DOSBOX-0.74<\/p><\/blockquote>\n

7. Then there you will see a file named \u201cDOSBOX 0.74 Options\u201d click on it to open.<\/p>\n

<\/a><\/p>\n

8. It will open a text file in notepad. Then navigate to the last line of that file where it shows \u201c# You can put your MOUNT lines here\u201d.<\/p>\n

<\/a><\/p>\n

9. Add the following lines after that.<\/p>\n

mount c c:\/\/tasm<\/p>\n

c:\/\/<\/p><\/blockquote>\n

10. And save that file.<\/p>\n

<\/a><\/p>\n

11. And now open DOSBOX. You\u2019re done.
\n12. To run a TASM program copy your program file into the TASM folder which is under your C: drive.<\/p>\n

<\/a><\/p>\n

13. And now open DOSBOX. And navigate to the TASM folder by typing cd TASM<\/i> and hit enter.<\/p>\n

<\/a><\/p>\n

14. Now type tasm  <program_name>. Replace <program_name> with your actual program name. in my case, my program name is the timer so I will type tasm timer.asm<\/p>\n

<\/a><\/p>\n

15. After you hit enter. You will see an error, warning messages. If there isn\u2019t anything wrong with your program syntactically, you will see none there.<\/p>\n

<\/a><\/p>\n

16. And now link your program to that type tlink <program_name> <\/i>and hit enter. And now execute program by typing <program_name> only.
\n17. The program will start executing. To stop execution type exit and hit enter.<\/p>\n

So in this way you can install TASM on your system and execute x86 code on your machine. This tutorial is applicable to Windows 7\/8\/8.1\/10.<\/p>\n

If you face any problem while doing this, make sure that you hit the comment below with your problem.<\/p>\n
