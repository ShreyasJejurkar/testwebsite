As I am one of the contributor(Shreyas02<\/a>) at Microsoft Community<\/a>, I have seen so many users are facing problem with Microsoft Edge<\/a>.
\nMost of the user\u2019s problem are this.
\n\u2022 Microsoft Edge doesn\u2019t not open although if they click on tile.
\n\u2022 Microsoft Edge is missing.
\nAnd if you are facing this issue then this post is only for you to solve the problem.
\nAs Windows 10 comes with Microsoft Edge browser. It\u2019s not an easy task to reinstall Edge as like we reinstall other programs directly. There is another way to do that. We have to Re-register the Microsoft Edge browser and then reinstall it.
\nSo just follow the steps given below and get back that new generation browser Microsoft Edge.
\n1. Press Windows logo key + Q<\/strong>. In that search box type \u201cPowerShell\u201d (without quotes). After PowerShell shows in search result, right click on that and click on \u201cRun as administrative\u201d.
\n2. After PowerShell open type following command in that.
\n<\/p>\n

Get-AppXPackage -AllUsers -Name Microsoft.MicrosoftEdge | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register \u201c$($_.InstallLocation)\\AppXManifest.xml\u201d -Verbose}<\/p><\/blockquote>\n

<\/strong>
\n3. After typing full command just hit enter. This command will re-register and reinstall Microsoft Edge on your PC. <\/u>
\n4. Just reboot the PC to take effect. And then open Microsoft Edge. Done.<\/p>\n
