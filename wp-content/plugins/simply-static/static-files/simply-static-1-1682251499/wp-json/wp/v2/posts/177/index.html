Hello there. I am Shreyas. After a long time I am came back with a new troubleshooting post about Windows 8.1.<\/strong> I know Windows 10 has been launched but still I am going to tell you about 8.1
\nMostly when people upgrade from Windows 8 to Windows 8.1, they face one problem about crashing of Change PC settings<\/strong>. This happen mostly when something get wrong while upgrading Windows version<\/u>.
\nBut its so easy to solve this problem.
\nSo follow the steps and get rid from this frustrating problem.
\n1. Open Command Prompt <\/strong>(Administrative). You can do this by pressing Windows logo key+X simultaneously and then from that short menu click on Command Prompt (administrative).
\n2. In that CMD window. Type the following command carefully. <\/p>\n

powershell -ExecutionPolicy Unrestricted Add-AppxPackage -DisableDevelopmentMode -Register $Env:SystemRoot\\ImmersiveControlPanel\\AppxManifest.xml<\/strong><\/p><\/blockquote>\n

Actually this is a Powershell command<\/u>. This command will repair the component of “Change PC settings”. After full command execution the CMD will return to same directory of system.
\n3. There is no requirement of restarting PC, but I will suggest you to restart the computer once<\/em>. And then open “Change PC Settings”. You will notice the change.
\nThanks for reading. If any problem then ask in comment section below.
\nCheers……..<\/p>\n
